A basic web application built with **Node.js**, **Express**, and **EJS**. This project demonstrates simple routing, form handling, and dynamic page rendering.
---

## Installation

1. Clone the repo:
   
   ```
   git clone https://github.com/nicolemellett/05_thirsty_33828132.git
   ```
2. Go to project folder

   ```
   cd 05_thirsty_33828132
   ```
3. Install dependencies

   ```
   npm install
   ```
4. Start server

   ```
   node index.js
   ```
5. Open browser

   ```
   localhost:8000
   ```
